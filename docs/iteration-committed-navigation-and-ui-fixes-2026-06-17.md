# 迭代纪要：导航与 UI 已确认修复提交（2026-06-17）

## 本次保留的修复

- 导航栏基础行为
  - 面包屑文件夹名称可点击导航。
  - 左侧 `>` 使用左侧目录作为下拉来源，避免误展示右侧目录的子目录。
  - `>` 文本垂直居中。
  - 地址栏编辑态点击外部区域后回到面包屑显示态。
  - 向上按钮按真实路径向上，支持离开扫描树直到盘符根目录。
  - 重复导航同一路径时避免重复压入历史栈。

- Treemap 空态
  - 路径离开扫描树时清空 Treemap 并提示需要重新分析。
  - 扫描树内但无可绘制子项时显示独立空态，不再误提示“选择磁盘并开始扫描”。

- 扫描结果过滤
  - 磁盘根目录直属的 `System Volume Information` 在 fallback 枚举和 MFT 快速扫描路径中都会被过滤。

- 推荐面板底部命中
  - 分析 overlay 与空/错误 overlay 不再跨行覆盖底部统计栏。
  - 移除推荐列表底部额外 padding，减少透明区域拦截点击。

- 标题栏图标
  - 标题栏左侧图标改为白色软盘 glyph，提高辨识度。
