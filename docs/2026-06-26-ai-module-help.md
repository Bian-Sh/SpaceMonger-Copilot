# 2026-06-26 AI 模块说明能力补充

## 背景
用户希望磁盘管理 Copilot 能解释 APP 内每个模块的用途与用法：当用户问“xxx 有什么用 / 怎么用”时，AI 能直接告知，不应误触发扫描、推荐清理等状态变更动作。

## 本轮实现
- 在 Core Copilot 路由中新增 `ModuleHelp` 意图，用于识别模块说明类问题。
- 说明类问题走本地回答，不依赖 API Key，不调用 LLM，不生成一级确认卡片。
- 对以下磁盘空间管理相关模块提供按需说明：
  - 扫描/路径输入
  - Treemap（矩形图）
  - TreeView（文件树）
  - 推荐清理/分析
  - AI Chat / Copilot
  - 白名单/保护规则
  - 设置/API Key
  - 控制台/日志
- 说明类问题会抑制动作生成，避免“推荐清理有什么用？”被识别成推荐分析执行请求。

## 设计边界
- 不把模块说明放入全局 system prompt，继续保持按意图触发。
- 不扩大 AI 可调用范围；本轮只增加解释能力，不增加主题、捐赠、关于页等非磁盘管理控制能力。
- 设置/API Key、白名单等仅做“用途/怎么用”解释，不开放 AI 直接控制入口。
